git基础操作
王子豪
大家合作的根本

git init
git init命令创建一个空的Git仓库或重新初始化一个现有仓库。

git clone <url/>
克隆项目到本地

git add <file/>
添加文件或文件夹

git status
git status 命令用于显示版本库和暂存区的状态

git commit -m "<message/>"
为本次提交增加备注

git push <remote/> <branch/>
这将把当前分支的更改推送到指定的远程仓库(<remote/>)的指定分支(<branch/>)

git pull <remote/> <branch/>
拉取指定的远程分支

git branch
git branch命令用于列出,创建或删除分支

git branch <name/>
在git branch命令的帮助下创建一个新的分支

git checkout <branch/>
切换分支

git merge <branch/>
合并分支到主分支

git diff
对数据源的不同版本进行比较

git diff --staged
上述命令将显示已经分档的文件的变化

git reset <commit/>

git revert <commit/>

git fetch <remote/>
用于从远程仓库获取最新的提交历史和分支信息，但并不自动合并或修改本地代码。其中，<remote/>是远程仓库的名称，如 origin。
